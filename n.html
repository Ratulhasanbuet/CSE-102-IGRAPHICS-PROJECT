🚀 Game Flow (Sequential)

  1. 🎬 Intro Screen – “Press Enter to Begin”
  2. 🧑‍💻 Name Input – Type your name
  3. 📖 Rules & Win Conditions (logic-based)
  4. 🎛️ Difficulty Select – Easy/Medium/Hard
  5. 🧠 Maze Selector – Four maze layouts
  6. 🎨 Background Selector – Style your game theme
  7. 🎮 Game Begins – Pac-Man, pellets, ghosts
  8. 🍒 Fruit Bonus System – 16 types with effects
  9. 💔 Death/Win Handling – Score, celebration, respawn
  10. 🏆 Highscore Entry – Save name & score
  11. 📜 Credits & Tribute – Sincere acknowledgments
  12. ❌ Exit Screen – Confirm & quit

📋 Rules & Winning Logic

  • 🎯 Objective: Eat all pellets and fruits without dying
  • 🧠 Ghosts become smarter over time and by difficulty
  • ⚡ Power pellets make ghosts vulnerable
  • 🍒 Fruits trigger bonus score or extra lives
  • 🏁 Win: Clear maze | 💥 Lose: All lives gone

📂 Project Folder Structure

├── pacman.txt             # Core game engine
├── fruit/                 # 16 fruit images
├── ghosts/                # Sprite sheet and individual ghosts
├── mazeDesignImage/       # Visual style maps
├── screens/               # Menu, settings, credits, tribute
├── background/            # Theme selectors
├── player_name.txt        # Highscore database

🧠 Technical Highlights

  • Ghost pathfinding via scatter/chase logic
  • Dynamic pellet and fruit rendering with scoring logic
  • File I/O using `player_name.txt` to record highscores
  • Animated sprite switching using `ghostanimation[]`
  • Frame rate control, death sequences, game state toggling
  • All transitions managed via screen flags (`menuc`, `exitc`, etc.)
